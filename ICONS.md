# Icon Requirements

The Chrome Extension requires icon files in the following sizes:

## Required Icons
- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)  
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Icon Design Guidelines
- Use Georgian script character (ქ) combined with English (A)
- Georgian green theme (#8bc34a) with soft design
- Transparent background for better integration
- Simple, recognizable design that works at small sizes

## Temporary Solution
For development, you can:
1. Create simple colored squares as placeholder icons
2. Use online icon generators
3. Design custom icons using tools like Figma, Canva, or GIMP

## Creating Icons
You can create basic icons using any image editor:
1. Create a 128x128 canvas
2. Use a green background (#8bc34a) 
3. Add Georgian character 'ქ' and English 'A'
4. Save as PNG
5. Resize to create 48x48 and 16x16 versions

The extension will work without custom icons, but Chrome will show default placeholder icons.
