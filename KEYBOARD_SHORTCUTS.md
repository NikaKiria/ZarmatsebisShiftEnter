# Keyboard Shortcut Guide

Two powerful keyboard shortcuts for different conversion needs:

## Ctrl+Shift+X - Convert & Replace
The **Ctrl+Shift+X** (Cmd+Shift+X on Mac) shortcut converts Georgian text in place or to clipboard.

## Ctrl+Shift+F - Fix & Search  
The **Ctrl+Shift+F** (Cmd+Shift+F on Mac) shortcut converts clipboard text and opens Google search.

## How They Work

### Ctrl+Shift+X (Convert & Replace)
Intelligently detects where you are and what to convert:

**1. Text Fields (Recommended)**
- Focus on any text input with Georgian text
- Press **Ctrl+Shift+X**
- Text gets converted instantly in place
- Works on: inputs, textareas, contenteditable fields

**2. Selected Text** 
- Select Georgian text anywhere on a webpage
- Press **Ctrl+Shift+X**  
- Converted text gets copied to clipboard
- Shows notification with before/after

**3. Clipboard Conversion**
- Copy Georgian text to clipboard
- Press **Ctrl+Shift+X** anywhere
- Clipboard gets replaced with English version
- Great for Chrome address bar accidents!

### Ctrl+Shift+F (Fix & Search)
Always works with clipboard content:

**1. Copy & Search**
- Copy Georgian text to clipboard (from anywhere)
- Press **Ctrl+Shift+F**
- Text gets converted and Google search opens in new tab
- Perfect for research and quick lookups

## Chrome Address Bar Workflow

### Using Ctrl+Shift+X (Convert Method)
When you accidentally type in the address bar:
1. **Select all** the Georgian text (Ctrl+A)
2. **Copy it** (Ctrl+C) 
3. **Press** Ctrl+Shift+X
4. **Paste** the converted text (Ctrl+V)

### Using Ctrl+Shift+F (Search Method)  
When you want to search what you typed:
1. **Select all** the Georgian text (Ctrl+A)
2. **Copy it** (Ctrl+C)
3. **Press** Ctrl+Shift+F
4. **Google search opens** automatically in new tab

Or alternatively:
1. **Type "geo "** in address bar
2. **Paste** your Georgian text
3. **Select** conversion from suggestions

## Notifications

Both shortcuts show helpful notifications:
- ✅ **Success**: Shows before → after conversion
- 🔍 **Search opened**: Shows when Google search launches
- ℹ️ **Info**: Guidance when no Georgian text found
- 📋 **Clipboard**: Indicates when text copied to clipboard

## Benefits

### Ctrl+Shift+X Benefits
- **No "geo" keyword needed** - works everywhere
- **Faster than clicking** the lamp icon
- **Multiple modes** - handles different contexts automatically
- **Perfect for text editing** - converts in place
- **Great for address bar accidents** - via clipboard method

### Ctrl+Shift+F Benefits  
- **Instant research** - copies, converts, and searches in one action
- **No manual Google navigation** needed
- **Perfect for learning** - quickly look up Georgian terms
- **Efficient workflow** - copy → shortcut → results
- **New tab opens** - doesn't disrupt current page

## Which Shortcut to Use?

**Use Ctrl+Shift+X when:**
- Typing in forms/inputs and made a mistake
- Want to convert text for copy/paste elsewhere  
- Need to fix text in Chrome address bar
- Converting for use in another application

**Use Ctrl+Shift+F when:**
- Want to search/research Georgian terms you see
- Found Georgian text you want to understand
- Need quick technical/shopping/educational lookup
- Want automated Google search workflow

Both shortcuts are designed for those "oops, wrong keyboard layout" moments, but serve different end goals! 🚀
