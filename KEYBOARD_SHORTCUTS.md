# Keyboard Shortcut Guide

The **Ctrl+Shift+X** (Cmd+Shift+X on Mac) shortcut is the fastest way to convert Georgian text to English!

## How It Works

The keyboard shortcut intelligently detects where you are and what to convert:

### 1. **Text Fields** (Recommended)
- Focus on any text input with Georgian text
- Press **Ctrl+Shift+X**
- Text gets converted instantly in place
- Works on: inputs, textareas, contenteditable fields

### 2. **Selected Text** 
- Select Georgian text anywhere on a webpage
- Press **Ctrl+Shift+X**  
- Converted text gets copied to clipboard
- Shows notification with before/after

### 3. **Clipboard Conversion**
- Copy Georgian text to clipboard
- Press **Ctrl+Shift+X** anywhere
- Clipboard gets replaced with English version
- Great for Chrome address bar accidents!

## Chrome Address Bar Workflow

When you accidentally type in the address bar:

1. **Select all** the Georgian text (Ctrl+A)
2. **Copy it** (Ctrl+C) 
3. **Press** Ctrl+Shift+X
4. **Paste** the converted text (Ctrl+V)

Or alternatively:
1. **Type "geo "** in address bar
2. **Paste** your Georgian text
3. **Select** conversion from suggestions

## Notifications

The shortcut shows helpful notifications:
- ✅ **Success**: Shows before → after conversion
- ℹ️ **Info**: Guidance when no Georgian text found
- 📋 **Clipboard**: Indicates when text copied to clipboard

## Benefits

- **No "geo" keyword needed** - works everywhere
- **Faster than clicking** the lamp icon
- **Multiple modes** - handles different contexts automatically
- **Perfect for accidents** - especially in address bar
- **Visual feedback** - clear notifications show what happened

This shortcut is designed specifically for those "oops, wrong keyboard layout" moments! 🚀
