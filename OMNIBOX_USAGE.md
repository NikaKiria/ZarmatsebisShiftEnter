# Chrome Search Bar Usage

The extension now works with Chrome's address bar using the **"geo"** keyword!

## How to Use

1. **Click on Chrome's address bar** (or press Ctrl+L)
2. **Type "geo"** followed by a space
3. **Type your Georgian text** (e.g., "geo ჰელლო ჟორლდ")
4. **Select from suggestions**:
   - 🔄 Convert and copy  
   - 🔍 Search with English version
   - 🌐 Navigate to URL (if applicable)

## Examples

### Basic Conversion
```
geo ჰელლო ჟორლდ
→ Shows: "hello world"
```

### Search Intent  
```
geo რა არის რეაქტ
→ Searches: "what is react"
```

### Website Navigation
```
geo ფეისბუკ.ღომ  
→ Navigates to: "facebook.com"
```

## Tips

- **Fast access**: Use Ctrl+L then type "geo "
- **Multiple options**: Choose convert, search, or navigate
- **Same accuracy**: Uses identical keyboard mapping as webpage detection  
- **Works offline**: Conversion happens locally

## Limitations

- Requires the "geo" keyword (Chrome security restriction)
- Cannot auto-detect Georgian text in address bar
- Only works when extension is installed and enabled

This feature is especially useful for quick searches when you accidentally type with Georgian keyboard layout!
